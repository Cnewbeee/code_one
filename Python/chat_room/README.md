# 简易 Python Socket 聊天室

这是一个基于 Python `socket` 和 `threading` 模块实现的简易多人聊天室。

## 功能特点

*   **多用户支持**：服务器支持多个客户端同时连接。
*   **消息广播**：任何客户端发送的消息都会被广播给其他所有在线用户。
*   **系统通知**：新用户加入或现有用户离开时，系统会自动通知所有人。
*   **昵称设置**：连接时可自定义昵称。
*   **健壮性**：包含基本的错误处理（连接断开、服务器未启动等）。

## 文件结构

*   `server.py`: 聊天室服务器端代码。
*   `client.py`: 聊天室客户端代码。

## 使用说明

### 1. 启动服务器

**场景 A：本地/局域网测试**
在本地电脑运行：
```bash
python server.py
```

**场景 B：公网服务器部署**
将 `server.py` 上传至公网服务器（IP: 124.222.4.182），并在服务器端运行：
```bash
python server.py
```
*注意：请确保服务器的防火墙/安全组已开放 TCP 12345 端口。*

### 2. 启动客户端

在任意电脑上运行客户端：

**Python 客户端**
```bash
python client.py
```

**Java 客户端**
```bash
java ChatClient
```

### 3. 使用流程

1.  客户端启动后，会提示输入服务器 IP。
    *   默认已设置为 `124.222.4.182`，直接回车即可连接到公网服务器。
    *   如果是本地测试，请输入 `127.0.0.1` 或局域网 IP。

    ```
    请输入服务器IP (默认 124.222.4.182): 
    ```
2.  连接成功后，输入昵称即可开始聊天。

    ```
    [连接] 正在连接到服务器 192.168.1.5:12345 ...
    [成功] 已连接到服务器！
    请输入您的昵称: 
    ```
3.  输入昵称并回车，即可进入聊天室。
4.  在聊天室中输入消息并回车，消息将发送给所有人。
5.  输入 `quit` 或 `exit` 退出聊天室。

## 配置

如果需要修改端口，请修改 `server.py` 和 `client.py` 文件顶部的配置项：

```python
PORT = 12345        # 服务器端口
```
