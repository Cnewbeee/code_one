# 简易 Python Socket 聊天室

这是一个基于 Python `socket` 和 `threading` 模块实现的简易多人聊天室。

## 功能特点

*   **多用户支持**：服务器支持多个客户端同时连接。
*   **消息广播**：任何客户端发送的消息都会被广播给其他所有在线用户。
*   **系统通知**：新用户加入或现有用户离开时，系统会自动通知所有人。
*   **昵称设置**：连接时可自定义昵称。
*   **健壮性**：包含基本的错误处理（连接断开、服务器未启动等）。

## 文件结构

*   `server.py`: 聊天室服务器端代码。
*   `client.py`: 聊天室客户端代码。

## 使用说明

### 1. 启动服务器

首先，在一个终端窗口中运行服务器：

```bash
python server.py
```

服务器启动后会显示本机的局域网 IP 地址，例如：
```
[启动] 服务器正在运行
[信息] 本机局域网IP: 192.168.1.5
[信息] 监听端口: 12345
[等待] 等待客户端连接...
```

请记下这个 IP 地址，其他电脑连接时需要用到。

### 2. 启动客户端

在另一台电脑（或本机）的终端窗口中运行客户端：

```bash
python client.py
```

### 3. 使用流程

1.  客户端启动后，首先会提示输入服务器 IP：
    ```
    请输入服务器IP (默认 127.0.0.1): 
    ```
    *   如果是本机测试，直接回车即可。
    *   如果是局域网其他电脑，请输入服务器显示的 IP 地址（如 `192.168.1.5`）。

2.  连接成功后，输入昵称：
    ```
    [连接] 正在连接到服务器 192.168.1.5:12345 ...
    [成功] 已连接到服务器！
    请输入您的昵称: 
    ```
3.  输入昵称并回车，即可进入聊天室。
4.  在聊天室中输入消息并回车，消息将发送给所有人。
5.  输入 `quit` 或 `exit` 退出聊天室。

## 配置

如果需要修改端口，请修改 `server.py` 和 `client.py` 文件顶部的配置项：

```python
PORT = 12345        # 服务器端口
```
